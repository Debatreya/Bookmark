# MyLMS Setup

### Frontend Setup instruction

1. Clone the project

    git clone https://github.com/Debatreya/MyLMS.git

2. Move into the directory

    cd client

3. install dependencies

    npm i

4. run the server

    npm run dev

### Backend Setup instruction

1. Clone the project

    git clone https://github.com/Debatreya/MyLMS.git

2. Move into the directory

    cd server

3. install dependencies

    npm i

4. run the server

    npm run dev
